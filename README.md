# Liver Disease Patient Data Analysis

## Project Overview
Analysis of 19,368 liver disease patient records using Excel, SQL and Tableau.
Built as a 2nd Semester Mini Project for the Department of Computer Engineering.

## Tools Used
- Microsoft Excel – Data cleaning and statistical analysis
- MySQL – Database queries and aggregation
- Tableau Public – Interactive dashboard

## Dataset
Indian Liver Patient Dataset (ILPD)
- 19,368 patient records
- 11 columns (age, gender, 7 blood tests, disease label)
- 71.3% patients have liver disease, 28.7% are healthy

## Key Findings
- ALT and AST enzyme levels are the strongest indicators of liver disease
- Both male and female patients have the same disease rate (71.3%)
- The 45–59 age group has the highest number of disease patients
- Albumin levels are lower in disease patients confirming reduced liver function
